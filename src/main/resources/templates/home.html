View Swagger documentation here: SWAGGER

View API Docs (JSON) here: API DOCS

To add seed data, place your events JSON file inside main/resources folder and name it data.json. After that, use the API (Swagger) to submit a POST request to the '/reload' endpoint.

NOTE: if the date last modified for the file has not changed, the new data might not be picked up.
